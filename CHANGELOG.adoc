== Release Process

1. Update version in `mix.exs` and `package.json`
2. Update this CHANGELOG
3. Create git tag: `git tag -a v0.1.0 -m "Release v0.1.0"`
4. Push tag: `git push origin v0.1.0`
5. CI creates GitHub/GitLab release
6. Publish to Hex.pm: `mix hex.publish`
7. Publish to npm: `npm publish`

== Version Scheme

- **MAJOR**: Breaking changes to CRDT wire format or API
- **MINOR**: New features, new CRDT types, backward-compatible
- **PATCH**: Bug fixes, documentation, performance improvements
